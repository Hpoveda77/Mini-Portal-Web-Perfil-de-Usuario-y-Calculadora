# Mini Portal Web

Mini proyecto web desarrollado con **HTML, CSS y JavaScript**.

## Descripción
Aplicación que permite navegar entre diferentes vistas usando un **iframe**:
- Perfil de usuario
- Calculadora básica

## Tecnologías
- HTML
- CSS
- JavaScript

## Funcionalidades
- Navegación interna con botones
- Visualización dinámica en iframe
- Operaciones básicas en calculadora (+, -, *, /)

## Vista
Proyecto sencillo con diseño moderno inspirado en el estilo de anime de naruto
